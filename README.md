# Automatic Water Planting System

A C++ based automatic water planting system project.

## Features

- Automated water distribution control
- Sensor-based moisture detection
- Real-time monitoring capabilities
- Efficient resource management

## Technologies Used

- **Language:** C++
- **Platform:** Cross-platform

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KODAVATISANJAY/automatic-water-planting-system.git
   ```

2. Navigate to the project directory:
   ```bash
   cd automatic-water-planting-system
   ```

3. Compile the source code:
   ```bash
   g++ test.cpp -o water_system
   ```

4. Run the program:
   ```bash
   ./water_system
   ```

## Usage

The system monitors soil moisture levels and automatically triggers water distribution when needed. It is designed for efficient irrigation management in agricultural and garden applications.
